The favicon is based on the jPulse Framework logo jpulse-logo.svg
Favicon generator: https://realfavicongenerator.net/

- favicon.svg: Static, simple format (circle + path, no mask/theme) for favicon generators and browser tab. Always blue circle, white pulse.
- jpulse-logo.svg: Theme-aware in-app logo; mask for transparent outside; light = primary circle + white wave, dark = reverse (light circle + darker blue wave). Use in pages where theme/color-scheme applies.
